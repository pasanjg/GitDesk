## Install

Clone the repo and install dependencies:

```bash
$ git clone https://github.com/pasanjg/GitHub-Desktop.git
$ cd GitHub-Desktop
$ npm install
```

## Starting Development

```bash
npm start
```

## Build Release

To package apps for the local platform:

```bash
npm run build
```
